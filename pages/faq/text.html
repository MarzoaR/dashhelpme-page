Faq

Dash dispone de dos billeteras que funcionan con SMS, las cuales permiten el envío y recepción de fondos sin la necesidad de internet. Ellas son Dash Text y CoinText.

El uso de Dash Text no requiere de ningún registro previo, tan solo basta con enviar un SMS con el comando correspondiente según la acción que deseamos realizar a los siguientes números 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos.

Para crear la billetera de Dash Text debe enviar el comando “CREAR” al 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos. Luego, el sistema le responderá notificándole que ha creado su billetera de Dash Text exitosamente.

Para enviar fondos, debe enviar el comando “ENVIAR”, el monto a enviar (ya sea en DASH, Bolívares, Pesos Colombianos o Dólares Estadounidenses) y el número telefónico (solo entre usuarios Dash Text) o address al 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos. Posteriormente el sistema le solicitará que confirme la transacción, una vez que haya verificado que los datos sean los correctos debe enviar “SI” al 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos.

Para recibir fondos en su billetera de Dash Text, solo debe enviar el comando “RECIBIR” al 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos. El sistema responderá de vuelta con su address o dirección y con su número de teléfono asociado a Dash Text (válido solo entre usuarios de Dash Text) para que pueda reenviarlo al usuario que realizará el pago.

Para consultar sus fondos disponibles en su billetera de Dash Text, debe enviar el comando “SALDO” al 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos. Acto seguido, el sistema enviará un mensaje de vuelta con su saldo disponible reflejado tanto en DASH como en Bolívares.

Para realizar una compra con Dash Text solo debe ingresar el comando "Compra" seguido del código que aparece en la pantalla del establecimiento comercial en donde va a adquirir el producto.

Para consultar la tasa de Dash actualizada, puede enviar tanto el comando “TASA” como el comando “PRECIO” al 3457 o al 9910 para Venezuela, al 899979 para Colombia, al +34931071060 para España y al 607-307-DASH para Estados Unidos. Posteriormente, el sistema le enviará el precio del Dash actualizado reflejado tanto en Bolívares como en DASH.
